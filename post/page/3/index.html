Sensr.net Patched Against Shellshock Bug

  • September 26, 2014
  • Adam Beguelin

Just a quick note to let you know that Sensr.net has been patched against the shellshock bug.  We run mostly CentOS servers which were vulnerable to the bug.  Our crack devops team patched all these servers on September 24, 2014.

Drop us an email if you have any questions.
