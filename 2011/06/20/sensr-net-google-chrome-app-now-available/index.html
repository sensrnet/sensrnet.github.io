First version of the Sensr.net Chrome App

Google’s Chrome web browser supports extensions and apps. We’ve started experimenting with this and today published a Sensr.net app in the Google Chrome Web Store.  This is our first attempt at a Chrome app, so it’s pretty simple and just takes you to the camera’s on Sensr.net that you’re allowed to view.

What would you like to see an app like this do in the future?  Let us know and we’ll try to make it happen.
