Using MyDLink and Sensr.net Together

  • August 9, 2011
  • Robert Whitefield

MyDLink allows anyone with select D-Link network cameras to monitor their cameras remotely from an internet-connected computer, iPhone, iPad, and Android device for free. The web interface makes it easy to check your network camera’s status on the site, configure camera settings, manage multiple netcams under a single account, and watch live video with sound, though this will soak up a bit more bandwidth.

While MyDLink is loaded with numerous excellent features by itself, most notably being able to watch your camera’s video with audio, there are several great reasons to use Sensr.net alongside D-Link’s proprietary web software.

Watching live video for longer than a minute requires Universal Plug n Play (UPnP) to be enabled on your router. So if it’s not turned on, you’re stuck refreshing the site every minute to continue watching live video. Sensr.net allows you to view unlimited live video from your IP camera from any location, as well as on iPhone/iPad and Android devices, even behind a firewall.

Setting up the Foscam FI8909W on Windows

  • August 3, 2011
  • fdegiuli

Foscam’s FI8909W is a relatively inexpensive IP camera that offers a variety of features, including built-in motion detection and fairly good night-vision capabilities. It comes with a handy little CD-ROM which contains the user guide, installation manual and a program that detects all Foscam cameras on the network. As a bonus, the Foscam CD also includes setup software for Mac, and in a .jar file if you want it on your phone; a big step up from the Windows-only installers that abound in the IP camera world. Of course, it’s entirely possible to set up your camera without using the provided software, if you’re loath to install random bloatware on your machine. This blog post is here to guide you through whatever setup process you should decide to take.
