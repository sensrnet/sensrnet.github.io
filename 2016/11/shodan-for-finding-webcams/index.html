Shodan is an IoT search engine. It randomly connects to IP addresses on the internet and tries to talk to them. Most devices will return some kind of response, Shodan calls these banners. For instance, a web server banner looks like this:

I got this header by running curl -D head http://blog.sensr.net, which does an HTTP GET to the given URL and saves the headers in a file, in this case named head. Since http://blog.sensr.net is hosted at GitHub, we can see that they run their own customer server software.
