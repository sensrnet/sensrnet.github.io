D-Link 931L

The D-Link 931L is the little brother of the D-Link 933L. While the 931L still has many of the functions of the 933L, including the wifi repeater which greatly increases it’s range, it does not have night vision. While this has some obvious security limitations it also means the 931L is $30 cheaper (the 931 currently goes for $39.99 on amazon). The best part about these two camera’s is they have the same setup and are both perfect for Sensr.net’s cloud recording system! If you want to learn how to setup the D-Link 931L or the DCS-933L just visit this tutorial here.
