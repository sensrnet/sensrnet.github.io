The Panasonic BL-C230A Wireless Internet Security Camera is a cute little PTZ (pan/tilt/zoom) camera. It works in wired and wireless mode. One feature that some of our users like is the Privacy button on top. It looks a bit like a snooze button on an alarm clock. When you push this button the camera’s LED turns from green to red.
