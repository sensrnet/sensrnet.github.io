How to setup the Tenvis JPT3815W

  • July 1, 2014
  • Geoffrey Pottow

You can record your Tenvis camera on Sensr.net with just a little bit of work. This blog will show you how to get it up and running with Sensr.net or any other service that supports FTP uploads of JPEGs.

The Tenvis JPT3815W is a great little IP camera with a ton of features that’s prefect for cloud recording. Some people complain that the user interface for the Tenvis products isn’t as sleek or easy to use as Foscam which may be true but it doesn’t really matter if you are using Sensr.net because our interface is breath taking no matter what camera you use!
