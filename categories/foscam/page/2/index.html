Setting up the Foscam FI8909W on Windows

  • August 3, 2011
  • fdegiuli
Tweet

Foscam’s FI8909W is a relatively inexpensive IP camera that offers a variety of features, including built-in motion detection and fairly good night-vision capabilities. It comes with a handy little CD-ROM which contains the user guide, installation manual and a program that detects all Foscam cameras on the network. As a bonus, the Foscam CD also includes setup software for Mac, and in a .jar file if you want it on your phone; a big step up from the Windows-only installers that abound in the IP camera world. Of course, it’s entirely possible to set up your camera without using the provided software, if you’re loath to install random bloatware on your machine. This blog post is here to guide you through whatever setup process you should decide to take.
